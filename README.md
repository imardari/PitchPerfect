# PitchPerfect

PitchPerfect is a fun app that records your voice and plays it back with various sounds effects!

## Features

<ul>
  <li>Record your voice</li>
  <li>Play the recording back with various sounds effects, which are:</li>
        -Slow
        -Fast
        -High Pitch
        -Low Pitch
        -Echo
        -Reverb
</ul>
 
## Requirements
iOS 11.0+
